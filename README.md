# Nepal TFlow: Tourism Flow Analysis System
Nepal TFlow is a smart tourism management platform designed to predict and analyze crowd movements using Time-Series Forecasting. It helps the tourism sector in Nepal make data-driven decisions to prevent overcrowding and improve visitor experiences.

# Tech Stack
**Frontend**: react and react-dom, react-router-dom, chart.js and react-chartjs-2, tailwindcss

**Backend**: Node.js / Express (or your specific backend)

**Database**: PostgreSQL / MongoDB (or your specific DB)

**Forecasting**: Python (Prophet/ARIMA) or specific time-series libraries

# Key Features
**Real-time Dashboard**: Overview of current tourist traffic across major spots.

**Predictive Analysis**: 7-day crowd forecasts using historical data.

**Interactive Heatmaps**: Visual representation of high-density areas.

**Automated Reporting**: Exportable PDF/Excel reports for stakeholders.

# Getting Started
To run this project locally, follow these steps:

**Clone the repository**
`git clone https://github.com/NilishaShrestha000/CAPSTONE.git`

**Install dependencies**
 `npm install`

**Run the development server**
`npm run dev`
